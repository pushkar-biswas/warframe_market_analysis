# warframe_market_analysis
This project aims at analyzing the market trends of in-game items of warframe using the API from warframe.market.

What is Warframe?
[WARFRAME](https://wiki.warframe.com/w/WARFRAME) is a 2013 free-to-play science fiction cooperative third-person shooter video game both developed and published by Digital Extremes

To understand warframe.market, first we need to understand what trading in Warframe?
In Warframe, the players, aka Tenno, fight enemies with weapons and abilities to complete mission objectives to earn items and resources.
These items and resources then can be used to upgrade the weapons and abilities to fight harder enemies and get better loot.
This game is a 'Looter-Shooter'. This cycle goes on until you go over the maximum possible damage value.

These items obtained by the player then can be traded between the players in the game.
Player can trade items for other items or for Platinum (in-game premium currency).

What does [Warframe.market](https://warframe.market/) does?
As the name suggests, It is a market place where player can create 'Buy order' and 'Sell order'. 
It is a convenient and organized place to by and sell items.

warframe.market provides API's for various functions of the site.

The aim of this project is to analyze the data from warframe.market and provide insights so that player can make more informed decision.
